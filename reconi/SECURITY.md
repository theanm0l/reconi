# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Reconi, please report it responsibly.

- **Email**: security@example.com
- **PGP Key**: [Link to PGP key]

Please do not open a public GitHub issue for security vulnerabilities.

## Scope

Issues in the following areas are in scope:
- Remote code execution in worker modules
- Credential leakage through logging or caching
- SSRF via URL fetching modules
- Insecure default configurations
- Dependency vulnerabilities

## Responsible Disclosure

We follow a 90-day disclosure timeline. Please allow this time for us to address the issue before publishing details.
